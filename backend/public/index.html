Fallback Page

This is a fallback page served by the backend. If you're seeing this, it means:

  1. The backend server is running correctly
  2. The frontend build files were not found in the expected locations

Troubleshooting

Here are some steps to fix this issue:

  1. Make sure the build process completed successfully
  2. Check that the frontend build files are in the correct location
  3. Verify that the backend server has access to the frontend build files

API Endpoints

The following API endpoints should be available:

  • /api/moderate-text - Text moderation API
  • /api/moderate-image - Image moderation API
  • /api/settings - User settings API
  • /api/auth - Authentication API
  • /health - Health check endpoint

Try accessing /health to verify the API is working.
